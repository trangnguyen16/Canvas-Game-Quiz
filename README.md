# Canvas-Game-Quiz
Introducing a new game where players can test their knowledge and earn points! The game features a variety of questions that players must answer correctly in order to accumulate points. With each correct answer, players will earn points that are tallied up throughout the game. Are you ready to put your knowledge to the test and see how many points you can earn? Join in and let's find out!

General knowledge: Scriptable, OOP method, arrays, loops, swaps, list, linked list.

Unity: UI canvas, text mesh, button.
